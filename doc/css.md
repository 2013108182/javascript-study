## 반응형웹 - Responsive Web

* 해상도에 맞게 사이트의 레이아웃을 변경해서 대응 하는 웹 개발 방식
* 미디어 쿼리를 사용 

```
    @media all and (min-width: 1140px) {
        nav ul li { float:left; }
    }
    @media all and (max-width: 1139px) and (min-width: 800px) {
        nav ul li { float:left; }
    }
    @media all and (max-width: 799px) {
        nav ul li { }
    }
```
